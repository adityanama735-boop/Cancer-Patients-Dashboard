This project presents a comprehensive analysis of cancer patient data using interactive Power BI dashboards. It is designed to uncover meaningful insights related to patient demographics, treatment patterns, and survival trends.

The project is divided into two interconnected dashboards that together provide a complete analytical view of cancer-related data.
🧩 Project Structure

This project consists of two related dashboards:

1️⃣ Cancer Patient Overview Dashboard
Focuses on patient demographics and distribution
Analyzes age groups, gender, and cancer types
Provides a high-level summary of patient data
2️⃣ Cancer Treatment & Survival Dashboard
Focuses on treatment patterns and outcomes
Analyzes survival rates and trends
Compares effectiveness across different treatments

👉 Both dashboards complement each other and together form a unified data analysis system.
